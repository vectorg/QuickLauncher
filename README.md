# QuickLauncher 启动器

## 功能简介
- 图标区域：可拖动排序、拖入可添加、批量勾选、全部启动、启动勾选、清空勾选、延迟依次启动、勾选时显示序号
- 命令区域：添加命令、启动命令、查看日志、查看相关脚本文件夹
- 日志记录、数据持久化

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行
```bash
python main.py
```

## 说明
- 拖入可执行文件到图标区域即可添加
- 勾选后依次显示序号，支持延迟依次启动
- 命令区域可添加任意命令，支持多选批量启动
- 日志和数据保存在当前目录
